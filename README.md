# Rainy Milk - 製品版

ミルキーフロッグの女の子「ミルカ」と過ごす、雨の午後のひとときを描いたノベルゲームです。

## 製品版の新機能
- **シナリオの大幅拡充**: マルチエンディング（全3ルート）を搭載。
- **セーブ・ロード機能**: 進行状況を最大5スロットまで保存可能。
- **バックログ機能**: 過去の会話をいつでも読み返せます。
- **設定メニュー**: テキスト表示速度や音量の調整が可能。
- **ビジュアルの刷新**: 高画質な背景、イベントCG、表情豊かなキャラクター立ち絵を追加。

## 遊び方
1. `index.html`をブラウザで開きます。
2. キーボード操作:
   - `Enter` / `Space`: テキスト送り
   - `S`: メニュー開閉
   - `L`: ロード画面
